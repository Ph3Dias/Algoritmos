# Introdução

Esse diretório contém alguns dos principais algoritmos ensinados na graduação em Ciência da Computação. Por enquanto, conta com apenas sete algoritmos de ordenação, sendo eles:  

* Bubble Sort
* Insertion Sort
* Selection Sort
* Merge Sort
* Quick Sort
* Counting Sort
* Radix Sort

A lógica de cada algoritmo é explicada em um arquivo README.md presente na pasta de cada algoritmo.

## Referências

CORMEN, Thomas H. et al. Introduction to Algorithms. 4. ed. Cambridge: MIT Press, 2022.
