# Introdução

Esse diretório contém alguns dos principais algoritmos ensinados na graduação em Ciência da Computação. Por enquanto, conta com apenas sete algoritmos de ordenação, sendo eles:  

* Bubble Sort
* Insertion Sort
* Selection Sort
* Merge Sort
* Quick Sort
* Counting Sort
* Radix Sort

A lógica de cada algoritmo é explicada em um arquivo README.md presente na pasta de cada algoritmo.

## Ordem de Estudo Recomendada

Para quem nunca viu nenhum desses algoritmos, recomendo seguir a seguinte ordem:

1. Ordenação em Tempo Quadrático

    1. Bubble Sort
    2. Selection Sort
    3. Insertion Sort
2. Ordenação em Tempo Quasilinear

    1. Merge Sort
    2. Quick Sort
3. Ordenação em Tempo Linear

    1. Counting Sort
    2. Radix Sort
